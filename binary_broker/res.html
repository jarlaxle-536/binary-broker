List of Foreign Currency Rates Today

Exchange rate is the value of one currency for the purpose of conversion to another. It is also regarded as the value of one country's currency in relation to another currency.

Both Reference exchange rates and middle-market exchange rates are based on ISO 4217:2015 (Codes for the representation of currencies), who specifies the format for a three-letter alphabetic code and three-digit numeric code assigned to each currency.
